# Chronus Portfolio

A dark, terminal-inspired portfolio site for Jon Marien. It highlights projects, security research, community work, and contact links in a compact editor-style layout.

The site is live at `chron0.tech`.

## What It Is

This repo contains the source for a personal portfolio built around the `chrono` identity. The design leans into a minimal terminal/editor aesthetic: dark surfaces, monospace headings, muted text, and purple accent links.

Content is organized so new projects, writeups, experience entries, and contact links can be added without rewriting the page structure.

## Main Sections

- About: short personal intro and current focus.
- Projects: featured projects plus an expandable archive.
- Security research: CVEs, CTF writeups, labs, and blog links.
- Community: hackathons, clubs, and event work.
- Contact: email and social links.
